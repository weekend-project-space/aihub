提示层 是第一个允许您跟踪和管理 GPT 提示工程的平台。PromptLayer 充当代码和 OpenAI 的 python 库之间的中间件。 PromptLayer 会记录您的所有 OpenAI API 请求，允许您在 PromptLayer 仪表板中搜索和浏览请求历史记录。
