RTutor 是一个基于 AI 的应用程序，可以快速生成和测试 R 代码。通过对OpenAI的Davinci（ChatGPT的兄弟）的API调用，RTutor将自然语言翻译成R脚本，然后在Shiny平台中执行。可以生成 R Markdown 源文件和 HTML 报告。 在这里查看 github 存储库：https://github.com/gexijin/RTutor
