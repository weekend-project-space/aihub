Face++ 人工智能开放平台是北京旷视科技有限公司（后称旷视科技）推出的、面向开发者的开放平台。Face++ 以 API 或 SDK 的形式，将领先的、基于深度学习的计算机视觉技术开放给开发者，帮助您的应用更好地读懂世界。
